# خليك رياضي - تطبيق التدريب أونلاين

## تشغيل المشروع محلياً
```
npm install
npm run dev
```

## رفعه على الإنترنت (Vercel) - أسهل طريقة

1. روح على https://vercel.com وسجل دخول (تقدر تستخدم حساب GitHub أو Google)
2. اختار "Add New Project"
3. ارفع المجلد ده كامل (أو اربطه بمستودع GitHub)
4. Vercel هيكتشف إنه مشروع Vite تلقائياً:
   - Build Command: `npm run build`
   - Output Directory: `dist`
5. دوس Deploy، وخلال دقيقة هيديك لينك مباشر زي:
   `https://khaleek-riyadi.vercel.app`

## رفعه على Netlify (بديل)

1. https://app.netlify.com
2. اسحب وأفلت مجلد المشروع، أو اربطه بـ GitHub
3. Build command: `npm run build` - Publish directory: `dist`

## ملاحظة مهمة عن البيانات
البيانات (الحسابات، التمارين، المتابعة) متخزنة محلياً في متصفح كل مستخدم (localStorage).
ده يعني:
- كل مستخدم بياناته محفوظة على جهازه/متصفحه بس
- لو فتح الموقع من متصفح أو جهاز تاني، هيحتاج يسجل من جديد
- لو عايز البيانات تكون موحدة لكل المستخدمين على أي جهاز (قاعدة بيانات حقيقية)، ده يحتاج خطوة إضافية (Backend) ممكن نضيفها بعدين.
